ParaRichesse Shop

Serverwide Shop is one of ParaRichesse's most main and essential features. It is the basis of ParaRichesse's Economy System.

LEARN MORE Get Started

ParaRichesse's Shop Module

The Shop is an Interactive Module, allowing Users to setup Shops in the form of menus, where users may purchase Items with Custom Actions. 

Shop Categories

Categories can be made to group Items together, allowing easier access.

Ease of Use

Complicated commands are replaced with Interactive Menus for the best experience possible!

Setup Your Shop

Steps to Setup and Personalize your own Shop!

1

Launch the Shop Module

Type !shop in any channel to receive a Direct Message from the bot, containing the Shop Menu. (Please ensure that the bot is able to DM you.)

2

Select "Edit Shop!"

Select "Edit Shop" to begin your journey of Shop Personalization!

3

Personalize!

Start adding Categories and creating Items! The possibilities are ENDLESS!

Shop Layout

Layout of the Shop to help you better Personalize it!

  • Category
  • Item
  • Actions

Name

Name of the Category. Can be Edited!

Description

Description of the Category. Can be Edited!

Emoji

Emoji which represents the Category, which is used for selecting the Category via the Shop Menu. Can be Edited!

Items

Categories can be configured with a set of Items.

Name / Desc / Emoji

Like Categories, the Names, Descriptions and Emojis of an Item may be Edited!

Price

Both the Price and Selling Price of the Item may be Edited!

Actions

Items can be configured with a set of Actions

Expiry

Items can be set to Expire after a Fixed Duration. 

Get-Role

Get a pre-existing Role assigned to the user.

Create-Role

Allow Users to Create their own Custom Roles.

Toggle

Actions can be Toggled on or off

Edit

Actions may be Created and Destroyed.
